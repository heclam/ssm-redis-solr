恩，现在就来讲一下这个项目吧<br/>
这个项目登陆采用shiro认证，使用solr建立索引库，这样以来方便检索，solr采用ik中文分词器，缓存使用redis，商品管理页面采用SpringMVC上传图片<br/>
环境搭建：<br/>
     JDK1.8<br/>
    tomcat9.0<br/>
    MySQL5.7<br/>
    shiro<br/>
    redis<br/>
    solr8.0<br/>
    前端采用Vue+element的方式布局页面<br/>
   下面来看一下展示的效果<br/>
